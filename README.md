# Symphony Lite API

Symphony Lite API is a scalable backend system built with Spring Boot that simulates a collaborative platform where users interact within spaces and events.

## 🚀 Features (Phase 1)

* Basic API setup
* Clean architecture structure
* User and Space domain modeling
* RESTful endpoints
* Docker-ready environment

## 🛠 Tech Stack

* Java
* Spring Boot
* PostgreSQL
* Docker
* Git

## 📦 Project Structure

Controller → Service → Repository → Database

## 🎯 Goal

To demonstrate backend engineering skills including clean code, scalability, and data-driven design.
